How to Use This Website

Getting Started

  1. Visit the Website: Open http://localhost:5000 in your browser
  2. Navigate Sections: Use the menu at the top to jump to different sections
  3. View Course Offerings: Check the Courses section to see available programs

Managing Students

  1. Load Students: Click the "Load Students" button to see all registered students
  2. Add New Student: Enter student name and course name, then click "Add Student"
  3. View Dashboard: Check the Dashboard section to see total student count

Technical Details

This application uses:

  • Frontend: HTML, CSS, JavaScript for user interface
  • Backend: Node.js with Express framework for API
  • Database: In-memory storage or MongoDB for data persistence
  • Communication: REST API calls via fetch() in JavaScript
